# my-store

## Онлайн магазин на Vue с реализацией отображения товаров, поиска, фильтрации и сортировки, а также корзиной товаров 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
